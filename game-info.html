Controls

  • Use the "W", "A" and "D" keys to move the ship
  • Press the right and left arrow keys or move the cursor to change the angle of the weapon
  • Click the icon in the top left or press M to open the game menu

Platform

  • Currently supports desktop (mobile version planned)
  • Tested for Chrome, Firefox and Edge browsers
  • Save data is stored in the browser. If browser history is cleared, save data may be lost
